## 开发

```powershell
npm install
npm run docs:dev
```

## 构建

```powershell
npm run docs:build
```

## 本地预览（构建产物）

```powershell
npm run docs:preview
```

## 目录

- `docs/`：文档内容
- `docs/.vitepress/config.mts`：站点配置（标题/导航/侧边栏等）
