# 8. 广播系统

2024年暑假，绵中对广播系统做了硬件升级，改动位于教室、寝室、食堂，新增花园、草坪音响。现将升级前后称作“旧系统”、“新系统”。

## 新系统

包括教室、寝室、食堂、草坪音响。均在两校区各自的广播室进行控制，应该是使用网络通信进行控制的，具体实现可能是1台设备1个IP，或一群功能相近的设备共用一个IP。

根据官网上的方案连接拓扑图，草坪、寝室音响在这方面的推断已得到证实。
![拓扑图](/img/Hushan.jpg)

由于教室广播音响在更换后也出现过不明原因电噪声，故可能连接过程中还涉及了向模拟音频信号的转换。

新系统使用了湖山音响全套方案，这是绵阳本地的品牌。高三食堂还安装了湖山牌功能不明的设备，可能是监测噪音的？

## 旧系统

称作“旧系统”，实际上此前教室与寝室的广播系统是分开控制的，并非一个系统。教室的音响收到的完全是模拟音频信号，两个音响对应两个信号源，一个信号源同一时间仅能广播一种信号。如高一的考试指令与高二的英语听力音频/歌曲曾不能同时播放，但两个信号源可同时工作。如开广播会时曾出现大音响放音乐、小音响放主持人声音的情况。若要阻止一个年级的广播在另一年级处响起，只能批量地断电。

山下寝室的广播只是简单的功放+喇叭，总控制设备在2号楼2F杂物间，其硬件还在，但现已停用，上次使用广播播通知可能是在疫情期间。

7、8、9号楼的广播总控在8号楼安全值班处，其安装时间较山下寝室晚，现在仍在工作，可能还是IP广播。（此外，为扼制"破鼎"现象的发生，山上的电铃在停用一年半后于2026年1月起重新启用）。

## 运作模式

广播系统翻新之前，教室铃声使用广播室电脑上"雨花石音乐打铃系统"定时打铃。翻新之后，使用湖山的广播控制软件控制打铃，考试时则会接上电脑音频，运行"雨花石"或四川高考专用的指令软件。

此外，寝室在早上、中午、下午、晚上均设置了音乐，起床时音乐不固定，偶尔会更换，睡前音乐一直为"First Sunlight"。另有上、下课铃提示睡觉、出寝室时间。山上3个寝室下午起床时会循环15遍令人诟病的起床号。

山下食堂和草坪一般同步播放音乐，以及接入"心晴FM"。歌单自2024年4月起未更换过。山上草坪音响会在同时段播放中国之声，食堂音响未工作过。

山上寝室还可以将U盘（FAT32单分区的）交给8号楼宿管，播放根目录下的歌曲，一般早上播放10分钟左右。
