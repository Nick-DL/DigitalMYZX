# 2. 教室电教设备（希沃）

## 发展沿革

与多数学校一样，绵中的教室电教设备也经历了从"角落电视（投影幕布）+主机"到"红外白板+投影仪+一体机"再到"一体机大屏"的发展历程。其中，早期的这两种设备都能在通慧楼5F生物实验室、实验准备室中见到。

## 希沃大屏

希沃大屏已覆盖绵中几乎所有教室（除改造成其他用途的教室以及通慧楼1F选修教室等）、教研室、阅览室、会议室以及部分阶梯教室。其中，大多数教室装配的是MT51主板（8代i7）的六代希沃，少数搭载更早版本（如网班教室）与更新版本（如阅览室为七代希沃）。2025年12月，通慧楼、远翔楼1F楼道下一共堆置了10套新版希沃（1件"主屏"+1件"副屏"），目前尚不清楚这些希沃会被装在哪里。

### 希沃管家（希沃集控）

希沃集控功能很多，但一些超标的，如希沃桌面、进程保护（杀进程锁屏）等未出现在绵中希沃上。在绵中运用较广的有冰点还原、视频广播、定时关机、视频巡课等。这些均以希沃管家及其插件为载体。

#### 冰点还原

冰点还原一直是令电教委员最为头疼的管家功能。这不仅限制了用武之地，由于许多电脑上锁时并未处于解决完所有问题的健康状态，一些重装了系统，没有装必要软件就被冻结的电脑更是令人抓狂，所以有时关闭冰点还原完全是合理正当的。

#### 破解方法

冰点还原破解方法。主要围绕获取管理员密码展开。

##### 获取管理员密码 <Badge type="info" text="失效" />

正规地关闭冰点还原需要绵中信息中心管理员的认证，方式包括扫码登录其希沃账号，以及输入管理员密码。

![输入密码](/img/HugoVerifyPassword.png)

此前，获得管理员密码的唯一方式是打开希沃管家目录中的"SeewoCore.ini"配置文件，并查看"pwd"对应的值，该值为管理员密码的MD5值。随着希沃管家更新，该文件中的关键部分逐渐改成"pwd2"值（无法破解），后被直接去除。通过安装旧版希沃管家，获取MD5值并前往解密查询网站进行查询，历届学长行到了"001234"、"myzx985211"两个早期密码。2024年暑假后再次更新密码，且不清楚是因为该密码过于复杂，还是因为新版希沃管家在生成MD5值时加了盐导致结果不同，该MD5值（`2db113f501e61fb0565513e3c64911e3`）无法在各大查询网站中查询，这种最直接的方法宣告失效。

##### 使用独立版冰点还原组件

::: warning
免责声明：在关闭冰点还原后请认真维护电脑并在问题解决后及时重新打开冰点还原。若电脑出现问题，本站不负任何责任。
:::

从原理上看，启用冰点还原后相关程序会修改系统底层，使开机后产生的文件更改不做实质上的存储。希沃管家在通过认证后执行相关程序开关冰点还原。于是，直接提取相关程序——"SeewoFreeze UI"及其配套组件并通过特定参数运行即可直接对冰点还原进行控制。（注：在运行之前需结束希沃管家的进程。）
![Unfreeze](/img/UnFreezeUI.png)
[查看具体操作方法](https://www.bilibili.com/video/BV1kg4y1d7MN/)

##### 使用插件 HugoAura 将破解功能注入希沃管家

希沃管家使用 Electron 开发，于是便有了漏洞与后门。HugoAura 将自身界面注入希沃管家主界面，进而屏蔽管理员密码等验证机制，实现各种修改，不仅仅开关冰点还原。在 1.5.5.3917，最新一版希沃管家的更新日志中提到修复了该插件的破解功能并附上了 GitHub 仓库的链接（这就是官方认证的含金量吗？！不过把仓库链接贴更新日志上又是何意味？想让更多人知道吗这是？而且似乎修复完还能用啊。）

#### 视频广播

一般用于家长会时校领导（党委书记）与年级领导（年级主任）的视频讲话。

#### 定时关机

该功能定时开关，打开时会在 7:30/8:00 时远程下达命令关机。若电脑执行关机的过程中通过"应用占用-取消关机"的方式阻止自动关机，几秒后系统会再次尝试关机。阻止关机可提前关闭希沃管家/断网。

以上两种功能均通过绵中校园网中部署的 AC 服务器（10.37.2.30）实现。

#### 视频巡课

各班班主任有本班教室的巡课权限，领导有各班巡课权限。巡课时，可查看前置视频画面、收取麦克风音频、查看屏幕内容，可向电脑屏幕投送跑马灯通知文字，也可直接连麦。调取敏感权限时，装有火线的电脑会弹框提示"media_capture.exe"正在调取敏感设备，摄像头启动时其左下角会亮起一个小的白色指示灯。

为避免神秘人的实践，可用粉笔涂上摄像头（记得擦掉），也可断网、禁用相关设备。

### 安卓系统

希沃在设计软件架构时考虑到了稳定性，大屏直接运行的是安卓系统，再通过视频信号连接、USB与触控的桥接运行内置主板上的 Windows 系统。当后者意外挂掉，用安卓系统还能实现白板、展台、打开文件等基础功能。

安卓系统默认禁止安装软件，可先点7次Android版本进入安卓P彩蛋，再点7次内核版本进入工程菜单，然后允许安装未知来源的软件。工程菜单中还可以修改开机动画（注：请替换经验证的动画文件，否则轻则不显示，重则直接变砖！），或进行工厂级还原（重置后需激活系统，可以在官网/公众号找希沃客服，也可使用算号器）等。

## 外围设备

### 展台

希沃展台，与大屏通过网线连接。

### 音响

通过大屏主板上的 AUX 输出连接功放，再连接音响。有数百毫秒的时延。许多教室的音响都会出现不稳定的电噪声，怀疑是主板漏电或音频接口接触不良所致。

### 智能笔

其本质为电容笔，但上了电池后可操控 PPT 翻页及音量。据称绵中早期为各班配备了此类智能笔，现在只提供不带电的普通电容笔。
