# 3. 教室灯

这里主要讨论2023年末各班教室逐步更换的最新一代教室灯，并安装的还有教室中控屏，可能还有走廊上安装的立达信网关。

## 灯

这一代灯色温偏暖（不可调），顶上透明能照亮天花板，中间有类似传感器的装置（功能不明），可手动用中控屏调整亮度。

## 中控屏

中控屏的基础功能包括控制灯开关、亮度，以及希沃开关，并提供了预设场景供一键操作。切换场景时，打开自动亮度的希沃会跟着调整屏幕亮度。

在关于预览查看IP、MAC地址、SN码等。设置需要管理员密码才能进入。

硬件上，支持Wi-Fi、PLC、ZigBee连接。每个中控都有一个Wi-Fi热点，推测是与设备通信的方式，密码未知。

安装在基座上时，中控屏与背部8针接口连接（网关），并同步更新时间、配置等。拆卸下来时可通过底部Type-C接口供电启动。连接上电脑时可被识别，设备属性显示"ADB设备"。adb能检测到该设备，然而adb shell中的指令几乎全部无法执行，读写文件（甚至/sdcard）也提示权限不足。据推测，该设备底层可能并非安卓，只是调试时引入了adb相关渠道。
